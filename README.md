# World Happiness EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on the World Happiness Report dataset to identify the factors influencing happiness across different countries.

## Dataset
- World Happiness Report (2019)
- 156 countries
- Features include GDP per capita, social support, life expectancy, freedom, generosity, and corruption perception.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
1. Data Loading and Exploration
2. Top 10 Happiest Countries
3. Bottom 10 Happiest Countries
4. GDP vs Happiness Score Analysis
5. Correlation Analysis
6. Factors Affecting Happiness Score

## Key Findings
- Finland is the happiest country in the dataset.
- GDP per capita has the strongest positive relationship with happiness.
- Social support and healthy life expectancy significantly affect happiness.
- Generosity has relatively low impact on happiness score.
